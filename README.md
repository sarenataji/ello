# Ello - Digital Detox Launcher iOS App

**A complete SwiftUI iOS app that reclaims your attention by removing visual triggers and gamifying focus.**

## 🎯 Purpose

In an era of "infinite scrolling" and notification overload, Ello reclaims your attention by removing visual triggers (colorful icons, badges, and complex layouts) that cause habitual phone usage.

## 📱 Core Features

### 1. Friction UI (Home Screen)
- Text-only list of essential tools
- Brain processes text slower than colorful images
- Adds "positive friction" to break "zombie-scrolling"

### 2. Gamified Focus XP System
- Ring in center tracks "Focus XP"
- Earn points for hours away from distracting apps
- Lose points if you "cheat" and open blocked apps
- Turns willpower into a visible, rewarding game

### 3. Active Shield (App Blocking)
- Tap "BEGIN FOCUS" to activate system-level block
- Blocks non-essential apps
- Intercepts with "Shield Screen" and "Quick Tips"

### 4. Pattern Interrupts (Quick Tips)
- Shows tips like breathing exercises or water reminders
- Replaces negative "No" with positive "Life Anchor"
- Helps reset mental state
